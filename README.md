# 1- E-Commerce Full Stack Design

> A pixel-perfect, Figma-inspired e-commerce frontend built with React. Focused on clean UI, smooth UX, and responsive design.

🚀 Live Demo: https://brand-ecommerce-site.netlify.app

### Key UI Components
- Navbar: Sticky header with search, cart, user icon
- Hero Section: Full-width banner with CTA
- Product Grid: 4-column responsive grid with hover effects
- Product Card: Image, price, rating, "Add to Cart" button
- Cart Sidebar: Slide-in cart with quantity controls
- Footer: Newsletter + social links
- 
### Tech Stack
- Framework: React 18 + Vite
- Styling: Tailwind CSS - Figma design tokens used
- Icons: Lucide React / React Icons
- Fonts: Same as Figma
- Animations: Framer Motion for smooth transitions
- Deployment: Netlify

### Design to Code Process
1. Figma Analysis: Studied the design system, components, and auto-layout structure
2. Pixel Perfect Implementation: Followed an 8px grid system for consistent spacing
3. Responsive Breakpoints: Matched Figma's mobile and tablet frames for all screen sizes
4. Color Palette: Added Figma variables directly into the Tailwind config
5. Typography Scale: Converted Figma text styles to match h1 to p tags
